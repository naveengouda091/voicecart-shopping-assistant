# 🎤 Voicekart – Voice-Based Shopping Chrome Extension

Voicekart is a Chrome extension that allows users to search, compare, and manage products using voice commands. It features a floating chatbot-style interface that makes online shopping faster, hands-free, and more interactive.

---

## 🚀 Features

### 🎙️ Voice Search
Search for products using natural voice commands.

### 🧠 Smart Query Processing
Converts voice input into meaningful product searches.

### 🛒 Cart Management
Add, view, and manage selected products easily.

### ⚖️ Product Comparison
Compare products within the cart or across platforms.

### 💬 Floating Chat UI
Clean and modern chatbot-style interface for smooth interaction.

### 🌐 Multi-Platform Search (Concept/Planned)
Fetch results from multiple e-commerce websites.

---

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Extension API:** Chrome Extensions API  
- **Voice Processing:** Web Speech API  
- **UI/UX:** Custom animations & floating UI design  

---

## 📦 Installation

1. Clone the repository:

        git clone https://github.com/naveengouda091/voicekart-shopping-assistant.git

3. Open Chrome and go to:

        chrome://extensions/

4. Enable Developer Mode (top right)

5. Click Load unpacked

6. Select your project folder

---

## 🎯 Usage

1. Click on the extension icon  
2. Open the floating chatbot interface  
3. Click the mic button 🎤  
4. Speak your product query (e.g., "Show me best budget headphones")  
5. View results and manage your cart
